# Steps (React Learning Project)

Ένα μικρό React project για εξάσκηση σε state και conditional rendering.

## Τι κάνει

Η εφαρμογή υλοποιεί ένα απλό “Steps” UI:

- Προβολή βημάτων (1–3) με ενεργό step
- Μήνυμα ανά βήμα
- Κουμπιά Previous/Next
- Toggle εμφάνισης του component

## Τι εξασκώ εδώ

- Function components
- State με `useState`
- Conditional rendering
- Basic styling με CSS

## Τεχνολογίες

- React
- Create React App
- CSS

## Scripts

Στον φάκελο του project:

- `npm start` → development mode
- `npm test` → test runner
- `npm run build` → production build
- `npm run eject` → eject (μη αναστρέψιμο)

## Σημείωση

Αυτό το project είναι εκπαιδευτικό και φτιαγμένο για practice στο React.
